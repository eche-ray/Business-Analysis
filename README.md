 **Electronics Retail Sales Analysis (Python)****

**Project Overview**
This project analyzes retail sales data from an electronics business to understand
sales performance, revenue trends, and regional contributions using Python.

The goal is to derive actionable business insights that can support decision-making
around product focus, regional strategy, and sales performance tracking.
 Dataset Description
The dataset contains 200 transaction-level records with the following columns:

- Order_ID
- Order_Date
- Product
- Category
- Region
- Quantity
- Unit_Price
- Revenue

**#Tools & Libraries Used**
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

# Analysis Performed
- Revenue by product
- Monthly sales trend analysis
- Revenue performance by region
- Product and category performance comparison

**#Key Insights**
- Certain products generate significantly higher revenue than others.
- Monthly sales show noticeable fluctuations over time.
- One region outperforms others in total revenue contribution.
- Accessories and core electronics show different performance patterns.
